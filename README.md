# 数据
## line.json
  - isShowAllDate：x轴是否显示当月全部日期，还是只显示开始日期、中间日期和结束日期 true:显示当月全部日期；false：显示固定的三个日期
  
  
# 折线图
- x轴可以显示固定日期，也可以显示全部日期
- 画笔颜色、渐变色可以设置改变

# 交叉图
- 可点击，交叉区域颜色可配
- 若不跨年，不显示年标签
- 若跨年，显示年标签
- 数据可以从左往右显示，也可以从右往左显示
